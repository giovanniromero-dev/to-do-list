# To Do List

[![JavaScript](https://img.shields.io/badge/JavaScript-ES6+-F7DF1E?logo=javascript)](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
[![CSS](https://img.shields.io/badge/CSS-3-1572B6?logo=css3)](https://developer.mozilla.org/en-US/docs/Web/CSS)
[![HTML](https://img.shields.io/badge/HTML-5-E34F26?logo=html5)](https://developer.mozilla.org/en-US/docs/Web/HTML)

A feature-rich vanilla JavaScript task management application with an intuitive interface for creating, organizing, and tracking daily tasks. Features persistent local storage, live date display, and smooth task reordering.

## Features

- **Task creation** with inline input and quick-add functionality
- **Completion state toggle** — mark tasks as done with visual feedback
- **Drag-and-drop reordering** for custom task prioritization
- **Persistent storage** via browser localStorage across sessions
- **Live date and time display** in the application header
- **Responsive design** — works on desktop and mobile browsers
- **Clean, minimal UI** with CSS-only animations and transitions

## Tech Stack

- **Vanilla JavaScript (ES6+)** — No frameworks, pure DOM manipulation
- **HTML5** — Semantic application structure
- **CSS3** — Custom styling with flexbox layouts and animations

## Getting Started

```bash
git clone https://github.com/giovanniromero-dev/to-do-list.git
cd to-do-list
# Open index.html in your browser
```

No build tools or dependencies required.
